# Urmia University LaTeX Assignment Template

This repository contains the official LaTeX assignment template designed for Urmia University, along with a fully implemented sample assignment.

The project follows a modular architecture separating structural definitions from stylistic configurations to ensure maintainability and extensibility.

##  Key Features

- Standardized university title page
- Structured section formatting
- Automatic numbering
- Persian language support (XeLaTeX)
- Modular class (`.cls`) and style (`.sty`) separation
- Professional academic layout

---

##  How to Use the Template

1. Download the `template/` folder.
2. Replace the content of `main.tex` with your assignment content.
3. Compile using **XeLaTeX**.
4. The output PDF will follow the standardized university format.

---

##  Sample Assignment

The `sample-assignment/` directory includes a complete implemented assignment demonstrating practical usage of the template, including structured sections, figures, and formatted content.

---

##  Intended Use

This template is developed for academic use at Urmia University and may be adapted for other departments with minimal modifications.
